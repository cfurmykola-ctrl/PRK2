 Interactive Knowledge Map
 Опис проєкту

Interactive Knowledge Map — це десктопний застосунок, розроблений з використанням технології WPF (Windows Presentation Foundation), який дозволяє працювати з ієрархічною структурою знань у вигляді тем і підтем.

Програма забезпечує можливість перегляду, пошуку, додавання, редагування та видалення інформації залежно від ролі користувача.

 Основні можливості

– перегляд тем і підтем у вигляді дерева;
– пошук по темах;
– додавання нових тем і підтем;
– редагування назв та описів;
– видалення тем;
– підтримка двох мов (українська / англійська);
– зміна теми інтерфейсу (світла / темна);
– розмежування доступу (студент / викладач);
– збереження даних у JSON-файл.

 Ролі користувачів
Студент

– перегляд тем;
– пошук інформації;
– доступ лише для читання.

Викладач

– додавання тем;
– редагування тем;
– видалення тем;
– додавання підтем.

 Технології

– C#
– WPF (Windows Presentation Foundation)
– XAML
– .NET
– JSON (для збереження даних)
– MVVM (архітектурний підхід)
– Git (контроль версій)

 Структура проєкту
PRK2/
│
├── Models/          # Моделі даних (Topic, User)
├── ViewModels/      # Логіка (MainViewModel)
├── Views/           # Сторінки (MainPage, ListPage, SettingsPage)
├── Services/        # Робота з JSON і налаштуваннями
├── Resources/       # Стилі та локалізація
├── topics.json      # Дані тем
├── settings.json    # Налаштування
 Запуск проєкту
Відкрити проєкт у Visual Studio 2022
Обрати конфігурацію Debug
Натиснути Start (F5)

 Збереження даних

– теми зберігаються у файлі topics.json
– налаштування (мова, тема) зберігаються у settings.json
